### Built With

* cypress
* node

### Prerequisites

* npm
* node  
  
### Installation

Clone the repo
   ```sh
   git clone https://github.com/jtf11235/amazonTask.git
   ```
Install NPM packages
   ```sh
   cd /your/project/path
   ```
   ```sh
   npm install
   ```
   
### Running Tests

Open cypress from ide terminal 
   ```sh
   npx cypress open
   ```
Select browser and run e2e test
